HOW TO USE

1. Drop new HTML files into the same folder.

2. Update manifest.json with:
{
    "title": "Module Name",
    "file": "your-file.html",
    "description": "Optional description"
}

3. Upload/push to GitHub.

4. Press "Refresh Directory" on the launcher page.

OPTIONAL FUTURE UPGRADE:
GitHub Actions can auto-generate the manifest.json file automatically.
